BitcoinIII Core
===============

https://bc3.network

For an immediately usable, binary version of the BitcoinIII Core software, see https://github.com/PinkStarrySky/BitcoinIII-Core/releases/tag/v29.1.0

What is BitcoinIII Core?
---------------------

BitcoinIII is a nearly 1:1 re-launch of the Bitcoin protocol.

The only difference between BitcoinIII and BitcoinII/Bitcoin is the block hash algorithm. Rather than using SHA-256d (double SHA-256), BitcoinIII uses SHA3-256t (triple SHA3-256) to produce the block header hash. This algorithm has the advantage of not (yet) having wide software/hardware support.

BitcoinIII Core connects to the BitcoinIII peer-to-peer network to download and fully
validate blocks and transactions. It also includes a wallet and graphical user
interface, which can be optionally built.
